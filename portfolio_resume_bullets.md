# Resume Bullets

## Program Finance Control Tower

- Built a program finance control dashboard using Kaggle scenario cost data; modeled EAC, ETC, CPI, cost variance, projected margin, and workstream watchlist status with automated KPI outputs and executive commentary.

## Budget vs. Actual FP&A Variance Analysis

- Developed a budget-vs-actual FP&A reporting package that reshaped raw budget and actual data into account-level variance reporting, forecast accuracy indicators, material variance flags, and management-ready variance explanations.

## Defense Contractor Peer Benchmarking

- Created a manufacturer benchmarking model analyzing estimated category revenue, market share, eligible multi-year CAGR, geographic exposure, and downside/base/upside long-range planning scenarios.
