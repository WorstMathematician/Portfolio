# Data Note

Place the Kaggle file `Dynamic_Cost_Accounting_Dataset.csv` in `data/raw/` before running the analysis script.

The dataset is treated as synthetic scenario data for portfolio analysis. It is used to demonstrate program finance controls, EAC and ETC modeling, cost performance indicators, margin analysis, and executive reporting.

The cleaned working package contains the raw file, output CSVs, and PNG visuals. If this GitHub folder does not include the raw data or images, use the downloadable project package from the portfolio build session.
