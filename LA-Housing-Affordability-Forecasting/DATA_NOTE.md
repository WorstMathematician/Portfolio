# Data Note

This project was built from a complete local project package containing:

- `acs_income_zcta_2015_2024.csv`
- `Zip_zori_uc_sfrcondomfr_sm_month.csv`
- `Complete Project.ipynb`
- `la_county_2025_risk_choropleth.html`
- `LA_Housing_Affordability_Presentation.pptx`
- `LA_Housing_Affordability_Written Report.pdf`

The GitHub connector used for this upload is unreliable with large and binary files, so the repository version includes a portfolio-ready summary, result tables, and a reusable pipeline skeleton rather than the full raw data, PDF, PowerPoint, and HTML map.

## Data sources

- American Community Survey 5-year median household income data, table B19013, by ZCTA for 2015 to 2024.
- Zillow Observed Rent Index ZIP-level monthly rent data, filtered to Los Angeles County and aggregated to annual rent estimates.

## Large local artifacts

The complete local project package should be retained separately because it contains the full notebook, raw data, report, presentation, and interactive choropleth map.
