# Data Note

Place the Kaggle file `Defense_manufacture_companies_2005_to_2024.csv` in `data/raw/` before running the analysis script.

The dataset is treated as synthetic scenario data for portfolio analysis. It is used to demonstrate manufacturer benchmarking, market concentration analysis, peer growth analysis, and long-range planning scenarios.

The cleaned working package contains the raw file, output CSVs, and PNG visuals. If this GitHub folder does not include the raw data or images, use the downloadable project package from the portfolio build session.
