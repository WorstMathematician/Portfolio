# Caffeine Hypothesis Test

## Project overview

This project performs a one-sample Z-test to evaluate whether the average caffeine content in a cup of coffee differs from the advertised value of 120 mg.

## Analytical value

The project demonstrates hypothesis testing, statistical decision-making, data visualization, and interpretation of p-values. It is useful as a portfolio project because it shows how to test a business or quality-control claim using sample data.

## Questions answered

- Is the sample mean meaningfully different from the claimed mean of 120 mg?
- What is the Z-statistic for the sample?
- Does the p-value support rejecting the null hypothesis?
- How should the result be communicated in plain language?

## Method

- Test type: One-sample Z-test
- Sample size: 50
- Claimed population mean: 120 mg
- Sample mean: 120.58 mg
- Z-statistic: 0.2734
- Two-tailed critical value at alpha = 0.05: plus or minus 1.9600
- P-value: 0.7845
- Decision: Fail to reject the null hypothesis
