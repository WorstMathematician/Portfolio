# Finance Portfolio Projects

This section contains three analytics projects for portfolio review.

## Projects

- Program Finance Control Tower
- Budget vs. Actual FP&A Variance Analysis
- Defense Contractor Peer Benchmarking

Each project includes a README, analysis code, KPI definitions, scorecards, output tables, and an executive summary.

## Data note

The datasets are Kaggle community datasets used as scenario data for portfolio analysis. They are not presented as audited financial statements, official records, or proprietary company financials.
